Description

A builder for monoidal values in a monadic context.

This module contains the Builder type that is used for building the Env, but also for the underlying Vars. And it can be used for any Monoidal value, and in any Monadic context (an Applicative context is often enough).

IsString

This module also exports an IsString instance for Builders, which is useful when building Strings or other types with an IsString instance.

For example, if we have a builder

stringBuilder :: Applicative m => Builder m String

then we can compose it with a String literal:

stringBuilder <> "some string" :: Applicative m => Builder m String

For this you must enable the OverloadedStrings language extension, by adding the following to the top of your file:

data Builder m a Source #
A builder for monoidal values in a monadic context.
Builders are Monoids if a is a Monoid and m is an
 Applicative, and can be composed with (<>). Builders are
 also Monads if m is a Monad.
