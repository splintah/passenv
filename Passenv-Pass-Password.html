data PasswordFile Source #

A PasswordFile is a file that contains a password on the first line. The rest of the contents will not be parsed.

This file format can be parsed using the passwordFile parser.

Example

myverygoodpassword
key: value
Foo
# Bar

Parsing this file will return:

PasswordFile
  { pwFilePassword = "myverygoodpassword"
  }
